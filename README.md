
# Настройте свою виртуальную среду для Windows

```bash
python -m venv env 
.\env\Scripts\activate 
cd app
pip install -r requirements.txt 
```
# Если вы используете Linux или macOS, вы вместо этого запустите

```bash
python3 -m venv env 
source ./env/bin/activate
cd app
pip install -r requirements.txt
```
